So you don't repeat them

Write State to Files Immediately
Context compaction can wipe memory mid-task. If it matters, write it down.
üîÄ
Sequential Browser Agents Only
Parallel agents share one browser = conflicts. Run one at a time.
üìÅ
One Workspace, One Truth
Dual workspaces = lost work. We lost 8 hours of improvements. Pick one.
üé®
Never Change HTML + CSS Together

üìé
Create Shareable HTML Files
